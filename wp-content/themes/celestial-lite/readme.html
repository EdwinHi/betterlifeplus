Celestial WordPress Theme

Introduction to Celestial Lite

Celestial is designed with a clean business style that is capable of handling almost any kind of business or personal website. You will find that Celestial is not only flexible, but it's scalable, and will last you a very long time. One of the big bonuses of Celestial (the free version) is that I've included the Twitter Bootstrap framework for responsive mobility on your favourite devices like the iPhone, iPads, Android phones, and others.

Recommended Plugins

Although not required, Celestial uses several plugins that helps you setup and manage your website or blog when using this theme. The demo site for Celestial uses these plugins:

  1. Advanced Text Widget: http://simplerealtytheme.com/plugins/atw-pro/
  2. Breadcrumb NavXT: http://mtekk.us/code/breadcrumb-navxt/
  3. Cleaner Gallery: http://justintadlock.com/archives/2008/04/13/cleaner-wordpress-gallery-plugin
  4. Contact Form 7: http://contactform7.com/
  5. Dynamic Widgets: http://www.qurl.nl/dynamic-widgets/
  6. Lightbox Plus: http://www.23systems.net/plugins/lightbox-plus/
  7. Responsive Slider: http://alienwp.com/plugins/responsive-slider
  8. Simply Exclude: http://www.codehooligans.com/projects/wordpress/simply-exclude/
  9. Special Recent Posts Free Edition: http://wordpress.org/extend/plugins/special-recent-posts/
  10. TinyMC Advanced: http://www.laptoptips.ca/projects/tinymce-advanced/
  11. WP Google Fonts: http://adrian3.com/projects/wordpress-plugins/wordpress-google-fonts-plugin/

The best way to find and install these is to log into your WordPress dashboard and then go to Plugins >> Install New >> search with the plugin name and then click to install from there. If you are unable to do this, then you can visit each of the plugin links above, download the zip files for each one, and then manually upload each one to the /wp-content/plugins/ directory. Then go into your admin area and activate your newly installed plugins.
