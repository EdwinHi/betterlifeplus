===========
ABOUT Theme
=========== 

Pinblack is a fun and functional theme ideal for photo galleries and portfolio sites. It supports custom menus and is widget ready.

This theme is compatible with Wordpress Version 3.4.

All the image graphics and icons bundled with this theme are created by the author and licensed under the GPL.

For free themes support, please contact us http://wpthemes.co.nz/contact/
